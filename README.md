# Projeto-Ferramenta-de-controle-de-investimentos
Este projeto utiliza planilhas do Excel para criar uma ferramenta que proporciona ao usuário uma visão clara do potencial de retorno de determinado investimento.

## PatriSim:
O nome é apenas uma junção da palavra Patrimônio e divideno. Como estes dois tipos de dado são um dos principais motivos da planilha existir, fiz uma junção entre elas que pareceu legal, então utilizei ele como nome deste projeto de Excel.

 ## Modo de usar:
 A primeira tabela é inserido o salário e o rendimento da carteira do usuário. Além disso, com base nos dados fornecidos pelo próprio usuário, a tabela retorna uma sujestão de quanto ele deve investir.

 A segunda tabela deve ser inserido nos campos: o quanto o usuário deseja investir; por quanto tempo ele pretende investir determinado valor; e a taxa de rendimento que é retornada mensalmente a ele. Então, com base nos valores dado pelo usuário, a tabela mostra o patrimônio acumulado e os dividendo retornados mensalmente investindo determinado valor por 5 anos.

 A terceira tabela é mais como uma projeção de cenários que mostra o patrimônio acumulado e o dividendo que pode ser arrecadado caso o usuário invista determinado valor pela determinada quantidade de anos.

 Por fim, a quarta e última tabela dá uma sugestão da porcentagem que o usuário pode dedicar (do total do valor que ele pretende investir) a determinada tipo de FII baseada em seu perfil, podendo ser: conservador, moderado ou agressivo.
